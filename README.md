# zomato-data-analysis-project

## Tools Use
- Python (Pandas, Matplotlib, Seaborn)
- Excel (Pivot Tables, Charts)
- Power BI (Dashboard)

## Project Workflow
1. Data Cleaning, EDA, and Visualization in Jupyter Notebook
2. Pivot Table Analysis and Dashboard Creation in Excel
3. Interactive Dashboard Creation in Power BI

## Key Features
- Data cleaning and preprocessing using Python
- Exploratory Data Analysis (EDA) with visualizations
- Pivot tables and charts in Excel for analysis
- Interactive dashboard built in Power BI

## Key Insights  
- Online ordering is widely adopted across restaurants  
- Ratings are relatively consistent across different locations  
- Cost patterns vary based on location and restaurant type

## Note
- The 'Rate' column was intentionally not cleaned in the notebook.
- It was handled separately in Excel and Power BI during dashboard creation.

## Files Included
- zomato.ipynb → Data cleaning, EDA, and analysis
- Power BI file (.pbix) → Final dashboard
- Excel screenshots → Pivot tables and charts
- Power BI screenshot → Dashboard preview
